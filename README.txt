## 🌐 Cómo usar el juego

Puedes:

- Abrir `index.html` directamente en tu navegador
- Subirlo a un servidor web
- Integrarlo en un LMS (SCORM compatible con ajustes adicionales)

---

## 💡 Recomendación

Para mejor compatibilidad de audio:
- Usa el juego desde un servidor o plataforma online
- Algunos navegadores limitan el audio en archivos locales

---

## 🎯 Créditos

Adaptación de un recurso original en Excel a experiencia interactiva web.
